## Quick Start
Your terminal must be in a directory called openCV_EYE. And before you start your "Raspberry Pi" MUST BE connected with webcam.</br>

```
pi@raspberrypi:~ $ cd openCV_EYE
pi@raspberrypi:~ $ python face_eye_detection.py
```
and, done.
